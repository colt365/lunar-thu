# 介绍 #

SmartMe是一款小巧的搜索引擎综合软件。通过SmartMe，您可以非常方便地获取多元的互联网信息。

只需要您输入简短的关键字，SmartMe就可以智能地帮助您得到：

  * google，百度，搜狗等流行搜索引擎的结果。
  * 来自维基百科专业的词条解释和相关链接。
  * 对于英文单词，提供来自Dict.CN的单词解释。

让您获得“信息尽在指间”的享受！
